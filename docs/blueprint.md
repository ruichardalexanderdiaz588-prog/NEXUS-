# **App Name**: Nexus

## Core Features:

- Age Verification: Age verification system to prevent minors from accessing adult content and adults from contacting minors. Includes tiered access with parental monitoring options.
- Profile Creation: Profile customization with options for gender identity, sexual orientation (public/private), interests, and bios. Includes hashtag support for content discoverability.
- Emotional Interface: Emotion-based UI theming.  Changes in user emotion detected through keyword analysis dynamically alters app color schemes and interface elements (avatar animations, etc.).
- Content Prioritization: Feed algorithm prioritizing posts that match user interests
- Community Moderation: Community moderation tools to report harassment, bullying, or inappropriate content.
- Avatar System: Z-Avatar creator to allow users to create stylized avatars, with system moderation to disallow violent content.
- AI Assistant: AI assistant (IMEA) to provide emotional support, detect policy violations, suggest breaks from the app if overuse is detected, and provide code unique to the user that provides easy access.

## Style Guidelines:

- Primary color: HSL(210, 70%, 50%) - A vibrant, calming blue (#3399FF).
- Background color: HSL(210, 20%, 95%) - A very light blue-tinted off-white for a clean and modern feel (#F0F8FF).
- Accent color: HSL(270, 60%, 60%) - A lively violet to draw attention to key interactive elements (#9966FF).
- Headline font: 'Poppins', sans-serif. Body font: 'PT Sans', sans-serif. A versatile and friendly combination that scales from titles to paragraph text. 
- Custom-designed icons for navigation and interactive elements. Use a flat design style with rounded corners to reflect the app's friendly vibe.
- Clean, modern layout with a focus on content and user interaction. Employ a card-based design for posts and a clear, consistent grid system for all views.
- Subtle animations to create a dynamic and engaging user experience, such as animated transitions and animated UI elements on user interaction. Animated avatars during IMEA support.
